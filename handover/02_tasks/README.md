# 업무 문서 목록

`02_tasks`는 **작업별 폴더**로 관리한다.
각 작업 폴더의 `README.md`에 작업 요약과 운영 기준을 적는다.

## 작성 원칙
- 1폴더 1업무
- 각 폴더의 메인 문서는 `README.md`
- 보조 자료는 같은 폴더 안에 `csv`, `md`, 스크립트 참고 문서로 관리
- 오래된 문서는 삭제하지 않고 `../99_archive/`로 이동

## 현재 작업
- `task-001-windows-프린터-출력/`
  - 파일 선택, 프린터 선택, 기본 출력 규칙 정리
  - 별칭 파일과 기본 프린터 규칙 파일 포함
