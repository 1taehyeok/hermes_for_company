# 파일 찾기 규칙

## 기준 경로
`\\192.168.1.63\어린이용품\(0)어린이 및 생활용품 역학\1. KC인증`

## 찾는 기준
1. 사용자가 준 **정확한 기준 폴더**가 있으면 그 경로를 먼저 사용한다.
2. 품목명 확인
3. 문서종류 확인
   - 분석노트
   - 갑지
4. **기준 경로 → 품목 폴더 → 해당 폴더 파일 목록 확인** 순서로 좁혀서 찾는다.
5. 기준 폴더가 이미 확정된 상태에서는 다른 폴더까지 넓게 검색하지 않는다.

## 분석노트/갑지 확인 절차
1. 폴더 안의 파일 목록을 먼저 확인한다.
2. 분석노트 요청이면 아래 우선 후보를 본다.
   - `시험·검사 기록표 붙임 ...`
   - 그 외 `붙임` 표기가 있는 파일
3. 갑지 요청이면 아래 우선 후보를 본다.
   - `시험·검사 기록표 갑지 ...`
4. 후보가 둘 이상이거나 파일명만으로 확정이 애매하면, 그 폴더 안에서 후보 파일의 형식(pdf/hwp)과 이름을 함께 보고 사용자 의도와 대조해 확정한다.

## 우선순위
1. **정확한 폴더 위치 확인**
2. 폴더 내 후보 파일명 확인
3. 문서 종류 확정
4. 출력에 더 안정적인 형식 선택
   - 기본 우선순위: pdf → hwp → excel

## 현재 확인된 어린이용 킥보드 예시
기준 폴더:
`\\192.168.1.63\어린이용품\(0)어린이 및 생활용품 역학\1. KC인증\2. 어린이제품\13. 어린이용 킥보드`

해당 폴더의 확인 파일:
- `시험·검사 기록표 갑지(최종) _어린이용 킥보드.hwp`
- `시험·검사 기록표 붙임 - P702-2-05(Rev.8).hwp`
- `시험·검사 기록표 붙임 - P702-2-05(Rev.8).pdf`
- `어린이용 킥보드 rawdata_2023_Rev 0.xlsm`
- `참고용.pdf`

해석:
- 이 폴더에서는 **분석노트 = `시험·검사 기록표 붙임 - P702-2-05(Rev.8)`**
- 이 폴더에서는 **갑지 = `시험·검사 기록표 갑지(최종) _어린이용 킥보드.hwp`**

## 메모
- 같은 문서라도 형식이 다를 수 있음
- hwp/excel은 필요 시 pdf 변환 후 출력 검토
- 실제 운영에서는 파일명에 `분석노트`가 없어도 `붙임`이 분석노트 역할일 수 있다
