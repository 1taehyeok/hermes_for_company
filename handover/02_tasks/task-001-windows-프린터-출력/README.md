# task-001 windows 프린터 출력

## 목적
분석노트, 갑지, 합리적 오용 같은 출력 문서를 찾아 출력한다.

## 문서 종류
- 분석노트
- 갑지
- 합리적 오용

## 실제 파일 형식
- pdf
- hwp
- excel

## 파일 선택 순서
1. 사용자가 **정확한 기준 폴더**를 주면 그 폴더부터 확인하고, 상위/다른 폴더로 넓히지 않는다.
2. `file-aliases.csv`에서 별칭 확인
3. alias가 있더라도 **출력 전에 해당 원본 경로의 실제 파일 존재 여부를 먼저 확인**한다.
4. alias 경로에 파일이 있으면 그 원본을 출력 대상으로 확정하고 진행한다.
5. alias 경로에 파일이 없으면 **기준 경로 → 품목 폴더 → 해당 폴더 파일 목록 확인** 순서로 다시 찾는다.
6. 재탐색으로 실제 파일을 찾았으면 `file-aliases.csv`를 업데이트해서 다음부터는 재탐색을 줄인다.
7. 분석노트/갑지 요청과 실제 파일명은 다를 수 있으므로 폴더 안의 실제 파일명을 확인한다.
8. 같은 문서라도 형식은 pdf/hwp/excel 중 하나일 수 있음

## 분석노트/갑지 식별 원칙
- 분석노트 요청은 실제 파일명이 `시험·검사 기록표 붙임 ...`일 수 있다.
- 갑지 요청은 실제 파일명이 `시험·검사 기록표 갑지 ...`일 수 있다.
- 파일명만으로 확정이 애매하면 **해당 품목 폴더 안의 후보 파일 목록을 먼저 확인**한 뒤 출력 대상을 확정한다.
- 기준 폴더가 이미 주어진 경우, 다른 폴더에서 비슷한 키워드 파일을 넓게 찾지 않는다.

## 프린터 선택 순서
1. 사용자가 프린터를 직접 지정하면 그 값을 사용
2. 없으면 `item-printer-defaults.csv`에서 품목별 기본 프린터 확인
3. 없으면 `x7600` 사용

## 사용 파일
- `document-types.csv` : 문서 종류
- `file-aliases.csv` : 파일 별칭
- `printer-aliases.csv` : 프린터 별칭
- `item-printer-defaults.csv` : 품목별 기본 프린터
- `file-search-rule.md` : 파일 찾기 규칙
- `명령어-템플릿.md` : 파일/프린터만 바꿔 쓰는 표준 명령어
- `acrobat-1회-검증-절차.md` : 중복/누적 의심 시 1회 검증 절차

## 작업 원칙
- 원본 삭제 금지
- 원본 직접 수정 금지
- `file-aliases.csv`에는 **원본 파일 경로**를 저장한다.
- alias는 파일을 빨리 찾기 위한 기준이며, 실제 출력 시 영문 작업본 생성 여부는 문서 형식/경로 조건에 따라 별도로 판단한다.
- 필요 시 작업폴더에서 복사본으로 처리
- PDF 출력에서 **한글 경로/파일명은 이슈로 간주**하고, 영문 작업본을 만들어 출력하는 것을 기본값으로 삼음
- HWP도 한글 경로/UNC 경로 리스크가 있으면 영문 작업본을 만들어 출력한다.
- 출력에만 쓰는 영문 임시 작업본은 작업 완료 후 삭제 가능
- `1장 출력`은 기본적으로 문서 전체 1부 출력으로 이해
