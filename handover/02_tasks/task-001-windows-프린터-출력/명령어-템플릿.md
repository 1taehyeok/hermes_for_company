# 명령어 템플릿

## 목적
파일 경로와 프린터명만 바꿔서 바로 사용할 수 있는 출력 명령어 규격을 남긴다.

## 공통 규칙
- 기본 실행 위치: WSL
- 실행 엔진: Windows PowerShell
- 바꿔 넣는 값은 우선 `파일 경로`, `프린터명`, 필요 시 `부수`, `페이지 범위`
- `1장 출력`은 기본적으로 **문서 전체 1부**
- PDF는 **원본이 한글 경로/파일명일 경우 영문 작업본을 만들어 출력하는 것을 기본값**으로 한다
- 영문 작업본은 `C:\Users\Public\Documents\hermes-print-jobs\YYYYMMDD_HHMMSS-...` 아래에 만든다
- 출력 전용 영문 임시 작업본은 작업 완료 후 삭제 가능
- 특정 페이지 출력이나 HWP처럼 작업복사본이 필요한 경우에만 스크립트 사용

## 1. PDF 전체 문서 출력 — 기본 템플릿
가장 먼저 쓰는 기본형.

### 1-1. 영문 작업본 만들기
원본이 한글 경로/파일명이면 먼저 영문 작업본을 만든다.

```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '$src="<원본파일경로>"; \
    $job="C:\\Users\\Public\\Documents\\hermes-print-jobs\\<작업폴더명>"; \
    $dest=Join-Path $job "<영문파일명>.pdf"; \
    New-Item -ItemType Directory -Path $job -Force | Out-Null; \
    Copy-Item -LiteralPath $src -Destination $dest -Force; \
    Get-Item -LiteralPath $dest | Select-Object FullName,Length,LastWriteTime | Format-List'
```

### 1-2. 영문 작업본 출력
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '$acro="C:\\Program Files\\Adobe\\Acrobat DC\\Acrobat\\Acrobat.exe"; \
    $file="<영문작업본경로>"; \
    $printer="<프린터명>"; \
    Start-Process -FilePath $acro -ArgumentList @("/t", $file, $printer)'
```

### 1-3. 작업 완료 후 영문 작업본 삭제
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '$file="<영문작업본경로>"; if (Test-Path -LiteralPath $file) { Remove-Item -LiteralPath $file -Force }'
```

### 바꿔 넣는 값
- `<원본파일경로>`: 원본 PDF의 한글/원본 경로
- `<작업폴더명>`: 예) `20260422_083923-kickboard-print`
- `<영문파일명>`: 예) `kickboard-analysis-note`
- `<영문작업본경로>`: 예) `C:\Users\Public\Documents\hermes-print-jobs\20260422_083923-kickboard-print\kickboard-analysis-note.pdf`
- `<프린터명>`: 예) `Samsung X7600 Series (10.69.16.2)`

### 예시
```bash

    New-Item -ItemType Directory -Path $job -Force | Out-Null; \
    Copy-Item -LiteralPath $src -Destination $dest -Force; \
    Get-Item -LiteralPath $dest | Select-Object FullName,Length,LastWriteTime | Format-List'
```

## 2. HWP 전체 문서 출력 — 기본 템플릿
한글 경로/UNC 경로 HWP는 **영문 작업본을 로컬 작업폴더에 만든 뒤** `HwpPrnMng.exe /p`로 보내는 것을 기본값으로 한다.

### 2-1. 영문 HWP 작업본 만들기
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '$src="<원본파일경로>"; \
    $job="C:\\Users\\Public\\Documents\\hermes-print-jobs\\<작업폴더명>"; \
    $dest=Join-Path $job "<영문파일명>.hwp"; \
    New-Item -ItemType Directory -Path $job -Force | Out-Null; \
    Copy-Item -LiteralPath $src -Destination $dest -Force; \
    Get-Item -LiteralPath $dest | Select-Object FullName,Length,LastWriteTime | Format-List'
```

### 2-2. 기본 프린터 확인
`HwpPrnMng.exe /p`는 기본 프린터를 타는 것으로 보고 먼저 확인한다.

```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  'Get-CimInstance Win32_Printer | Where-Object {$_.Default} | Select-Object Name,Default,PrinterStatus,WorkOffline | Format-List'
```

### 2-3. HWP 출력 제출
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '& "C:\\Program Files (x86)\\Hnc\\Office 2018\\HOffice100\\Bin\\HwpPrnMng.exe" /p "<영문작업본경로>"'
```

### 2-4. 직후 큐/로그 확인
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '"QUEUE:"; \
    Get-PrintJob -PrinterName "<프린터명>" -ErrorAction SilentlyContinue | \
      Select-Object ID,DocumentName,JobStatus,PagesPrinted,TotalPages,SubmittedTime | Format-Table -AutoSize; \
    "EVENTS:"; \
    Get-WinEvent -LogName "Microsoft-Windows-PrintService/Operational" -MaxEvents 10 | \
      Select-Object TimeCreated,Id,Message | Format-List'
```

### 바꿔 넣는 값
- `<원본파일경로>`: 원본 HWP의 한글/원본 경로
- `<작업폴더명>`: 예) `20260422_0950_kickboard-hwp`
- `<영문파일명>`: 예) `childrens-kickboard-analysis-note`
- `<영문작업본경로>`: 예) `C:\Users\Public\Documents\hermes-print-jobs\20260422_0950_kickboard-hwp\childrens-kickboard-analysis-note.hwp`
- `<프린터명>`: 예) `Samsung X7600 Series (10.69.16.2)`

### 예시
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '$src="\\192.168.1.63\어린이용품\(0)어린이 및 생활용품 역학\1. KC인증\2. 어린이제품\13. 어린이용 킥보드\시험·검사 기록표 붙임 - P702-2-05(Rev.8).hwp"; \

    Copy-Item -LiteralPath $src -Destination $dest -Force; \
    Get-Item -LiteralPath $dest | Select-Object FullName,Length,LastWriteTime | Format-List'
```


## 3. PDF 1회 안전 검증 출력 — 템플릿
중복/누적이 의심될 때만 사용.

### 순서
1. 큐 확인
2. Operational 로그 확인
3. 아래 명령 1회만 실행
4. 사용자 실물 확인

### 큐 확인
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  "Get-PrintJob -PrinterName '<프린터명>' -ErrorAction SilentlyContinue | \
    Select-Object ID,DocumentName,JobStatus,PagesPrinted,TotalPages,SubmittedTime | Format-Table -AutoSize"
```

### Operational 로그 상태 확인
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  "Get-WinEvent -ListLog 'Microsoft-Windows-PrintService/Operational' | \
    Select-Object LogName,IsEnabled,RecordCount,LastWriteTime | Format-List"
```

### 1회 출력
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '$acro="C:\\Program Files\\Adobe\\Acrobat DC\\Acrobat\\Acrobat.exe"; \
    $file="<파일경로>"; \
    $printer="<프린터명>"; \
    Start-Process -FilePath $acro -ArgumentList @("/t", $file, $printer)'
```

### 직후 로그 재확인
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  "Get-WinEvent -LogName 'Microsoft-Windows-PrintService/Operational' -MaxEvents 10 | \
    Select-Object TimeCreated,Id,LevelDisplayName,Message | Format-List"
```

## 4. 스크립트 기반 표준 출력 — 템플릿
작업폴더와 로그를 남겨야 하거나, 특정 페이지 / HWP 처리처럼 스크립트 경로를 써야 할 때 사용.

### PDF 전체 문서
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -ExecutionPolicy Bypass -File /home/hth/work/hermes-ops/scripts/print_windows_file.ps1 \
  -SourcePath '<파일경로>' \
  -PrinterName '<프린터명>' \
  -Copies 1
```

### PDF 특정 페이지
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -ExecutionPolicy Bypass -File /home/hth/work/hermes-ops/scripts/print_windows_file.ps1 \
  -SourcePath '<파일경로>' \
  -PrinterName '<프린터명>' \
  -Copies 1 \
  -PageRange '<페이지범위>'
```

### HWP 출력
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -ExecutionPolicy Bypass -File /home/hth/work/hermes-ops/scripts/print_windows_file.ps1 \
  -SourcePath '<파일경로>' \
  -PrinterName '<프린터명>' \
  -Copies 1 \
  -SetDefaultPrinterForHwp
```

### 바꿔 넣는 값
- `<파일경로>`: 예) `\\192.168.1.63\어린이용품\...\분석노트 - 최종.pdf`
- `<프린터명>`: 예) `Samsung X7600 Series (10.69.16.2)`
- `<페이지범위>`: 예) `1` 또는 `1-3`

## 5. 파일/프린터 존재 확인 템플릿

### 파일 확인
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  '$f="<파일경로>"; Test-Path -LiteralPath $f; Get-Item -LiteralPath $f | Select-Object FullName,Length,LastWriteTime | Format-List'
```

### 프린터 확인
```bash
/mnt/c/Windows/System32/WindowsPowerShell/v1.0/powershell.exe -NoProfile -Command \
  "Get-Printer -Name '<프린터명>' | Select-Object Name,DriverName,PortName,PrinterStatus | Format-List"
```

## 6. 권장 사용 기준
- **그냥 전체 PDF 1부 출력**: 1번 기본 템플릿
- **HWP 전체 출력**: 2번 HWP 기본 템플릿
- **중복/누적 의심**: 3번 1회 안전 검증 템플릿
- **로그/작업폴더 남겨야 함**: 4번 스크립트 템플릿
- **특정 페이지만 출력**: 4번 PDF 특정 페이지 템플릿

## 7. 현재 검증된 조합
- PDF: `analysis-note-final.pdf`
- 프린터: `Samsung X7600 Series (10.69.16.2)`
- 엔진: Acrobat `/t`
- 결과: 1회 정상 출력 확인
- 추가 확인: 어린이용 킥보드 PDF는 **한글 원본 경로보다 영문 작업본 경로에서만 정상 출력 확인**
- HWP: `childrens-kickboard-analysis-note.hwp`
- 엔진: `HwpPrnMng.exe /p`
- 조건: 원본 HWP를 `C:\Users\Public\Documents\hermes-print-jobs\...` 아래 영문 파일명 작업본으로 복사 후 제출
- 로그: `800 → 801 → 842 → 805 → 307`, 이벤트 `307`에서 3페이지 확인

